# 推送消息格式示例

## 优化后的消息格式

### 单个池子格式

```
🎯 *USDC/USDT*
🟡 BSC

```
APR:    125.50%
TVL:    $2.50M
Volume: $1.20M
Fees:   $500.00K
```

🔄 *交易对:* USDC / USDT

🔗 [查看详情 →](https://kyberswap.com/earn/pools/xxx)
```

### 多个池子格式

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
✨ *发现 2 个新池子*
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

┌─ *池子 #1* ─────────────────────
🎯 *USDC/USDT*

🔗 [查看详情 →](https://kyberswap.com/earn/pools/xxx)

⏰ 2026-01-09 20:30:45
```

## 格式特点

### ✨ 清晰的结构
- 使用分隔线（━━━）清晰划分区域
- 使用边框字符（┌─ ─── ─── ───）突出每个池子
- 使用代码块（```）展示核心数据，保持对齐

### 🎨 视觉层次
- **标题**：使用粗体标记重要信息
- **数据区**：使用代码块格式，数据对齐整齐
- **链接**：使用箭头符号（→）引导点击

### 📊 信息组织
1. **标题区**：显示池子名称和链信息
2. **数据区**：核心指标（APR、TVL、Volume、Fees）
3. **交易对**：清晰显示交易对信息
4. **操作区**：提供查看详情的链接
5. **底部信息**：显示监控来源和时间戳

### 🔍 链标识
- 🟡 BSC（币安智能链）
- 🔵 Base（Base 链）
- ⛓️ 其他链

## 首次运行格式

首次运行时会显示：

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🎉 *首次运行 | 发现 10 个池子*
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
...
```

## 格式优势

1. **易读性**：清晰的分隔和层次结构
2. **专业性**：使用表格样式展示数据
3. **美观性**：使用 Unicode 字符美化界面
4. **信息完整**：包含所有关键信息
5. **操作便捷**：直接提供查看链接

## 实际效果

在 Telegram 中，消息会以 Markdown 格式渲染：
- 粗体文本会加粗显示
- 代码块会以等宽字体显示，保持对齐
- 链接可以点击跳转
- 分隔线和边框字符会保持原样显示

这样的格式既专业又美观，便于快速浏览和理解池子信息！
